# Overview

This repository contains two playable educational game prototypes created as part of the Texas Sports Academy Web-Based Game Developer assignment.

The goal was to demonstrate:

* Rapid game prototyping
* AI-assisted development workflows
* Educational gameplay design
* Web deployment
* Ability to integrate mathematics into engaging game loops

---

# Playable Games

## 🏀 Dunk Digits (Via GODOT)

A basketball-themed educational arcade game.

### Core Loop

1. A math equation appears above the basket.
2. Three basketball answer choices appear.
3. The player selects the correct basketball.
4. The selected basketball shoots into the hoop.
5. Correct answers increase score and combo.

### Educational Focus

* Addition
* Multiplication
* Fast mental math
* Pattern recognition

### Design Goal

Create a game that feels like a basketball challenge first and a math exercise second.

---

## 👾 Mathomon (Web Based via Antigravity)

A monster-catching educational prototype.

### Core Loop

1. A wild monster appears.
2. The player solves math problems before the monster escapes.
3. Correct answers allow a capture attempt.
4. New monsters continuously appear.

### Educational Focus

* Fast arithmetic
* Mental calculation under pressure
* Decision making

### Design Goal

Combine creature-collection gameplay with educational mechanics while maintaining a fast and addictive gameplay loop.

# Development Approach

The project followed a rapid prototyping philosophy:

1. Generate game concepts.
2. Identify opportunities for educational integration.
3. Build the smallest playable version.
4. Iterate quickly.
5. Deploy to the web.

The focus was on demonstrating speed, creativity, and effective AI-assisted development rather than extensive content production or polish.
